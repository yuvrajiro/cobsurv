Installing cobsurv
==========================

The easiest way to install scikit-survival is using `pip`:

    pip install cobsurv

The current minimum dependencies to run scikit-survival are:

- joblib~=1.3.2
- pycox~=0.2.3
- scikit-learn~=1.2.2
- numba~=0.57.1
- numpy~=1.24.0
- pandas~=1.4.4
- scikt-survival
